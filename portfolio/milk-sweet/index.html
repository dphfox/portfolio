Professionally-designed set of 176 cursors for desktop UI. Constructed to be faithful to current universal cursor shapes, focusing on subtle design refinements, natural proportions, readability and consistency.

Cursors were drawn on a 32x32 pixel grid and tested at 96 DPI for clarity. Cursors are available in 4 themed sets covering the CSS specification for cursor designs, in both vector and raster formats.
