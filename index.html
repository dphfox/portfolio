About me

In brief

I'm a creator living in London, England. In my teens, I was inspired to build games using Roblox and Minecraft, and later experimented with Source engine mapping.

I now develop open-source libraries, independently build plugins & tools for game creators, and produce some videos about game development.

I hold a First-Class Honours degree in BSc Computing Sciences from the University of East Anglia, taken to study computer graphics and light transport.

Tools I use

I've used Roblox for 14 years. Nowadays, I also occasionally use Unity.

Figma is my tool of choice for UI design and mock-ups.

Blender is my 3D modelling workhorse.

Ableton Live 11 Suite is my digital audio workstation.

I use paint.net as a minimal & effective raster image editor.

Programming languages & packages

I professionally script in Luau. I previously used JavaScript with Node.js.

My systems work is confidently in Rust. I've also built a few things with C/C++.

wgpu is my go-to for GPU work. I know some Vulkan too.

I'm fluent and natural in CSS and HTML - hopefully you can tell!
